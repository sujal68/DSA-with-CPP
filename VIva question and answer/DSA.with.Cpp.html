üí† Question 10: What is Method Overloading and Method Overriding in OOPs?

Sir, dono concepts Polymorphism se related hain ‚Äî but inka use aur purpose alag hota hai. Method Overloading compile-time polymorphism ka part hai, jabki Method Overriding runtime polymorphism ka part hai.

1Ô∏è‚É£ Method Overloading (Compile-Time Polymorphism)

    cout << m.add(5.5f, 2.5f) << endl;

    return 0;
}
üöó Real Life Example:

Sochh tu ‚Äúcall()‚Äù function use kar raha hai phone me ‚Äî tu kisi ko number se bhi call kar sakta hai, contact name se bhi, aur shortcut se bhi. Function ka naam same hai, par input (parameters) alag hain. Wahi hota hai Method Overloading.

2Ô∏è‚É£ Method Overriding (Runtime Polymorphism)

Method Overriding tab hota hai jab child class apni parent class ke method ko same name aur parameters ke saath dobara define karti hai. Matlab base class ka method overwrite ho jaata hai aur jab object child class ka hota hai, to child ka version call hota hai.

    void sound() {
        cout << "Animal makes a sound" << endl;

    void sound() {
        cout << "Dog barks" << endl;
    }

In short, Method Overloading ek hi function name ke multiple versions banata hai (different parameters ke saath), jabki Method Overriding ek parent class ke function ko child class me redefine karta hai. Dono ka goal same hai ‚Äî code reusability aur flexibility.

üí† Question 11: What is Virtual Function and Pure Virtual Function in OOPs?

Sir, Virtual Function aur Pure Virtual Function dono runtime polymorphism ke concepts hain, lekin dono ka use alag purpose ke liye hota hai.

1Ô∏è‚É£ Virtual Function

Virtual function ek base class ka function hota hai jise hum child class me override kar sakte hain. Jab hum base class ka pointer ya reference use karke child class ke object ko access karte hain, to **runtime pe decide hota hai** kaunsa function call hoga (base ka ya child ka). Isse kehte hain dynamic binding.

üíª Example (C++): 
#include <iostream>
using namespace std;

class Animal {
public:
    virtual void sound() {
        cout << "Animal makes a sound" << endl;
    }
};

class Dog : public Animal {
public:
    void sound() {
        cout << "Dog barks" << endl;
    }
};

int main() {
    Animal *a;   // base class pointer
    Dog d;
    a = &d;
    a->sound();   // Runtime polymorphism - Dog's sound() called
    return 0;
}
üê∂ Real Life Example:

Sochh ek ‚ÄúRemote‚Äù (base class pointer) hai jisse tu TV, AC ya fan control kar sakta hai (child classes). Ab remote ka button same hai, par result object ke type pe depend karega ‚Äî TV on karega, ya AC ‚Äî ye runtime pe decide hota hai. Yehi concept hota hai Virtual Function.

2Ô∏è‚É£ Pure Virtual Function

Pure Virtual Function ek aisa virtual function hota hai jiska base class me koi definition nahi hota ‚Äî sirf declaration hoti hai. Ye function child class me **must override** karna padta hai, warna error aata hai. Jis class me pure virtual function hota hai, wo class abstract class kehlati hai.

üíª Example (C++): 
#include <iostream>
using namespace std;

class Shape {
public:
    virtual void draw() = 0;   // Pure Virtual Function
};

class Circle : public Shape {
public:
    void draw() {
        cout << "Drawing Circle" << endl;
    }
};

int main() {
    Shape *s;
    Circle c;
    s = &c;
    s->draw();   // Output: Drawing Circle
    return 0;
}
üé® Real Life Example:

Sochh ek ‚ÄúShape‚Äù class hai jisme sab shape ka common rule hai ‚Äî ‚Äúdraw()‚Äù karna, par har shape (circle, rectangle, triangle) ka draw karne ka tareeka alag hota hai. To base class sirf rule define karti hai (pure virtual function), aur har child class apna version likhti hai ‚Äî yehi abstraction aur polymorphism ka combo hai.

üßæ Difference Between Virtual and Pure Virtual Function:

Feature Virtual Function Pure Virtual Function
Definition Has definition in base class Has no definition in base class (only declaration)
Overriding Can be overridden Must be overridden
Class Type Normal class Abstract class
Object Creation Object can be created Object cannot be created
Use To achieve runtime polymorphism To achieve abstraction & enforce overriding

In short, Virtual Function ka matlab hota hai ‚Äî ‚Äúchild class function ko runtime pe call karna‚Äù, aur Pure Virtual Function ek rule set karta hai ke ‚Äúchild class apna version lazmi define kare‚Äù. Dono milke runtime polymorphism aur abstraction ka backbone banate hain.

üí† Question 12: What is an Abstract Class in OOPs?

Abstract class ek blueprint ki tarah hoti hai jisse hum dusri classes ko banate hain ‚Äî ye ensure karti hai ki har child class ek particular rule follow kare. Matlab, abstract class base structure provide karti hai, aur child class uska real implementation karti hai.

    void info() {

    }
};

// Derived Class
class Circle : public Shape {
public:
    void draw() {
        cout << "Drawing a Circle" << endl;
    }
};

// Another Derived Class
class Rectangle : public Shape {
public:
    void draw() {
        cout << "Drawing a Rectangle" << endl;
    }
};

int main() {
    // Shape s;  ‚ùå Error: cannot create object of abstract class
    Shape *s;
    Circle c;
    Rectangle r;

    s = &c;
    s->draw();  // Calls Circle's draw()

    s = &r;
    s->draw();  // Calls Rectangle's draw()
    return 0;
}

üí† Question 19: What is a Stack in Data Structure?

Sir, Stack ek linear data structure hai jo LIFO (Last In, First Out) principle pe kaam karta hai. Matlab jo element sabse last me insert hua hai, wo sabse pehle remove hota hai. Jaise hum plates ko ek ke upar ek rakhte hain ‚Äî jo plate sabse upar hai wo sabse pehle nikalti hai.

üîπ Stack ke Main Operations:

  • push(x): Add element on top of the stack.
  • pop(): Remove top element from the stack.
  • top(): Returns the top element without removing it.
  • isEmpty(): Checks whether the stack is empty.
  • isFull(): (in static implementation) Checks if stack is full.
üíª Example (Stack using Array in C++): 
#include <iostream>
using namespace std;

#define MAX 5

class Stack {
    int arr[MAX];
    int top;

public:
    Stack() {
        top = -1;
    }

    void push(int x) {
        if (top == MAX - 1) {
            cout << "Stack Overflow!" << endl;
        } else {
            arr[++top] = x;
            cout << x << " pushed to stack." << endl;
        }
    }

    void pop() {
        if (top == -1) {
            cout << "Stack Underflow!" << endl;
        } else {
            cout << arr[top--] << " popped from stack." << endl;
        }
    }

    int peek() {
        if (top == -1) {
            cout << "Stack is empty!" << endl;
            return -1;
        }
        return arr[top];
    }

    bool isEmpty() {
        return (top == -1);
    }
};

int main() {
    Stack s;
    s.push(10);
    s.push(20);
    s.push(30);
    cout << "Top element: " << s.peek() << endl;
    s.pop();
    s.pop();
    s.pop();
    s.pop(); // underflow
    return 0;
}

    int n2 = r - m;

    int L[n1], R[n2];
    for (int i = 0; i < n1; i++) L[i] = arr[l + i];
    for (int j = 0; j < n2; j++) R[j] = arr[m + 1 + j];

        if (L[i] <= R[j])
            arr[k++] = L[i++];
        else
            arr[k++] = R[j++];
    }
    while (i < n1) arr[k++] = L[i++];
    while (j < n2) arr[k++] = R[j++];
}

        mergeSort(arr, l, m);
        mergeSort(arr, m + 1, r);
        merge(arr, l, m, r);
    }
}

    int i = (low - 1);

    for (int j = low; j < high; j++) {
        if (arr[j] <= pivot) {
            i++;
            swap(arr[i], arr[j]);
        }
    }
    swap(arr[i + 1], arr[high]);
    return (i + 1);
}

void quickSort(int arr[], int low, int high) {

        int pi = partition(arr, low, high);
        quickSort(arr, low, pi - 1);
        quickSort(arr, pi + 1, high);
    }
}

üíª Example (Graph using Adjacency List): 
#include <iostream>
#include <vector>
using namespace std;

class Graph {
    int V;
    vector<vector<int>> adj;
public:
    Graph(int V) {
        this->V = V;
        adj.resize(V);
    }

    void addEdge(int u, int v) {
        adj[u].push_back(v);
        adj[v].push_back(u); // for undirected graph
    }

    void display() {
        for (int i = 0; i < V; i++) {
            cout << i << " -> ";
            for (auto node : adj[i]) {
                cout << node << " ";
            }
            cout << endl;
        }
    }
};

int main() {
    Graph g(4);
    g.addEdge(0, 1);
    g.addEdge(0, 2);
    g.addEdge(1, 3);
    g.display();
    return 0;
}
